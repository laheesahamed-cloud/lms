# Backend Workspace

This is the new NestJS backend for the LMS rebuild.

Main local command:

```bash
npm run start:dev
```

Build check:

```bash
npm run build
```

This API is expected to run on:

```text
http://localhost:3000/api
```

For the full local run flow, use the root [README.md](/Applications/XAMPP/xamppfiles/htdocs/lms/README.md).
